# Deloitte Australia Data Analytics Job Simulation

This repository contains my work completed as part of the **Deloitte Australia Data Analytics Virtual Job Simulation** on Forage.

## Overview

This simulation provided hands-on experience in solving real-world business problems using data analytics and forensic technology. The tasks focused on transforming raw data into actionable business insights through data classification, visualization, and analysis.

## Skills Demonstrated

- Data Analytics
- Data Cleaning
- Data Classification
- Business Intelligence
- Data Visualization
- Dashboard Design
- Business Insight Generation
- Microsoft Excel
- Tableau

## Tasks Completed

### Task 1 – Data Classification

- Classified employee equality scores using Microsoft Excel.
- Applied logical formulas to categorize equality scores into predefined classes.
- Ensured data consistency and accuracy for further analysis.

### Task 2 – Tableau Dashboard

- Imported and prepared the dataset in Tableau.
- Designed an interactive dashboard to visualize workforce equality data.
- Used charts and filters to present key business insights.
- Identified trends and patterns to support data-driven decision-making.
